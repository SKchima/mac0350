# Escalonador de Estações

## O que faz?
O Escalonador de Estações é uma aplicação para gerenciar reservas das máquinas na Casa do Código. As reservas são feitas sempre na semana seguinte, mas há também uma seção representando a própria sala da Casa do Código, em que um usuário pode entrar e usar uma estação desocupada.


## Como usar?
Para rodar o projeto, basta rodar o script <setup.sh> presente na raiz. Ele se encarregará de criar o servidor local, instalar as dependências e ligar o FastAPI.
